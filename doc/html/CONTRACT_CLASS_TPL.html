Macro CONTRACT_CLASS_TPL

CONTRACT_CLASS_TPL — Macro used to declare nested classes with contracts within a type-dependent scope (e.g., within templates).

Synopsis

// In header: <contract/class.hpp>

CONTRACT_CLASS_TPL(class_declaration)

Description

This macro is the exact same as CONTRACT_CLASS but it must be used when declaring nested classes with contracts within a type-dependent scope.
