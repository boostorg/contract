Class template old_ptr_noncopyable

boost::contract::old_ptr_noncopyable — Old value pointer (that does not require the pointed old value type to be copy constructible).

Synopsis

// In header: <boost/contract/old.hpp>

template<typename T> 
class old_ptr_noncopyable {
public:
  // types
  typedef T element_type;  // Pointed old value type. 

  // construct/copy/destruct
  old_ptr_noncopyable();
  old_ptr_noncopyable(old_ptr< T > const &);

  // public member functions
  T const & operator*() const;
  T const *const operator->() const;
  explicit operator bool() const;
};

Description

This is set to point to an actual old value via either BOOST_CONTRACT_OLDOF or boost::contract::make_old.

See Also:

Advanced Topics

Template Parameters

  1. typename T

    Type of pointed old value. If this type is not copy constructible, this pointer will always be null (but this library will not generate compile-time errors when this pointer is dereferenced).

old_ptr_noncopyable public construct/copy/destruct

  1. old_ptr_noncopyable();
    Construct this object as a null old value pointer. 
  2. old_ptr_noncopyable(old_ptr< T > const & other);
    Construct this object from an old value pointer of copyable-only types.

    This constructor is implicitly called by this library when assigning an object of this type using BOOST_CONTRACT_OLDOF.

    Parameters:

    other

    Copyable-only old value pointer.

old_ptr_noncopyable public member functions

  1. T const & operator*() const;
    Dereference this old value pointer.

    This will generate a run-time error if this pointer is null (but no compile-time error if the pointed type T is not copy constructible).

    Returns:

    The old value (contract assertions should not change the state of the program so this member function is const and it returns the old vale as a constant reference). 

  2. T const *const operator->() const;
    Structure-dereference this old value pointer.

    (This will not generate a compile-time error if the pointed type T is not copy constructible.)

    Returns:

    The old value (contract assertions should not change the state of the program so this member function is const and it returns the old value as a constant pointer to a constant object). 

  3. explicit operator bool() const;
    Check if this old value pointer is null or not (safe-bool operator).

    Returns:

    True if this pointer is not null, false otherwise.
