Class old_value

boost::contract::old_value — Convert user expressions in old values.

Description

This class is often only implicitly used by this library and it does not explicitly appear in user code.

On older compilers that cannot correctly deduce the boost::is_copy_constructible trait, programmers can manually specialize that trait to make sure that only old value types that are copy constructible are actually copied (see the boost::is_copy_constructible documentation).

Advanced Topics

old_value public construct/copy/destruct

  1. template<typename T> 
  old_value(T const & old, 
            typename boost::enable_if< boost::is_copy_constructible< T > >::type * = 0);
    Construct this object from the specified old value when the old value type is copy constructible.

    The specified old value is copied (one time only) and the related old value pointer will be set to not null (no copy is made instead if postconditions are not being checked, see BOOST_CONTRACT_NO_POSTCONDITIONS).

    Parameters:

    old

    Old value to be copied.

    Template Parameters:

    T

    Old value type. 

  2. template<typename T> 
  old_value(T const & old, 
            typename boost::disable_if< boost::is_copy_constructible< T > >::type * = 0);
    Construct this object from the specified old value when the old value type is not copy constructible.

    The specified old value cannot be copied in this case so it is not copied and the related old value pointer will always be null.

    Parameters:

    old

    Old value (that will not be copied in this case).

    Template Parameters:

    T

    Old value type.
