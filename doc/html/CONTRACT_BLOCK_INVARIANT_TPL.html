Macro CONTRACT_BLOCK_INVARIANT_TPL

CONTRACT_BLOCK_INVARIANT_TPL — Macro used to specify block invariants within a type-dependent scope (e.g., within templates).

Synopsis

// In header: <contract/block_invariant.hpp>

CONTRACT_BLOCK_INVARIANT_TPL(assertions)

Description

This macro is the exact same as CONTRACT_BLOCK_INVARIANT but is must be used when specifying block invariants within a type-dependent scope.
