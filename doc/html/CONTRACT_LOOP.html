Macro CONTRACT_LOOP

CONTRACT_LOOP — Macro used to declare a loop that will specify a loop variant.

Synopsis

// In header: <contract/loop_variant.hpp>

CONTRACT_LOOP(loop_declaration)

Description

This macro must be used to declare a loop that will later specify a loop variant using the CONTRACT_LOOP_VARIANT macro.

Parameters:

loop_declaration This is a loop declaration (for, while, etc) that follows the usual C++ syntax (see also the Grammar section).
